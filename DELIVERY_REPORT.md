## 📝 実装メモ

### 技術スタック
- **HTML5**: ページ構造とCanvas要素
- **CSS3**: レスポンシブデザイン、グラデーション背景、アニメーション
- **JavaScript(ES6+)**: ゲームロジック、イベント処理
- **Canvas API**: 2D描画エンジン

### ゲーム仕様
1. **プレイヤー(自機)**
   - 画面下部に配置された緑色の戦闘機
   - 左右移動と弾発射が可能
   - 連射制限(300msクールダウン)実装
   - 被弾でライフ減少(初期ライフ3)

2. **敵(インベーダー)**
   - 5行×11列の隊列配置(合計55体)
   - 行ごとに色分け(赤→オレンジ→黄→緑→シアン)
   - 左右移動+端で反転+徐々に下降
   - ランダムに弾を発射
   - 上の行ほど高得点(50点→40点→30点→20点→10点)

3. **バリケード(遮蔽物)**
   - 4つのバリケードを配置
   - 自機弾・敵弾の両方で徐々に破壊される
   - 体力に応じて色が変化(緑→暗緑)

4. **当たり判定**
   - AABB(Axis-Aligned Bounding Box)方式
   - プレイヤー弾 vs 敵
   - プレイヤー弾 vs バリケード
   - 敵弾 vs プレイヤー
   - 敵弾 vs バリケード

5. **ゲームフロー**
   - タイトル画面 → ゲーム画面 → 結果画面(ゲームオーバー/クリア)
   - リトライ機能実装
   - タイトルへ戻る機能実装

### 操作方法
- **PC**: ← →キーで移動、Spaceキーで発射
- **スマホ**: 画面左側タップで左移動、右側タップで右移動、中央タップで発射

### UI/UX
- レトロなグリーンモノクロームデザイン
- タイトルのパルスアニメーション
- ボタンホバーエフェクト
- スコア・ライフのリアルタイム表示
- レスポンシブデザイン(モバイル対応)
